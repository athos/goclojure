# goclojure
[![Clojars Project](https://img.shields.io/clojars/v/goclojure.svg)](https://clojars.org/goclojure)

`goclojure` brings `goruby`-like abbreviation to Clojure.

`goruby` is a Ruby implementation specialized to code golf.
